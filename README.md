#  Conversor de Moedas em Portugol

Projeto simples desenvolvido em Portugol para conversão entre Real e Dólar.

##  Funcionalidades

- Conversão Real → Dólar
- Conversão Dólar → Real
- Suporte a valores decimais (com ponto)

##  Objetivo

Projeto criado para praticar pseudocódigo, lógica de programação e entrada de dados.
